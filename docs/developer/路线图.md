# 路线图 🗺️

本文档概述 Heablcoin 项目的未来发展方向和计划。

---

## 🚀 近期目标 (2025年Q1)

### 交易所支持扩展
- [ ] 集成 OKX API 支持
- [ ] 集成 Bybit API 支持
- [ ] 统一的交易所适配器接口

### 回测引擎
- [ ] 实现基于历史 OHLCV 数据的本地回测框架
- [ ] 支持策略绩效可视化（权益曲线）
- [ ] 回测报告生成

### 通知渠道扩展
- [ ] Telegram Bot 集成
- [ ] Discord Webhook 集成
- [ ] 企业微信/钉钉推送

---

## 🌟 中期目标 (2025年Q2)

### Web UI 仪表板
- [ ] 开发独立的 React/Next.js 前端仪表板
- [ ] 实时监控活跃策略
- [ ] 可视化持仓和盈亏变化

### 高级策略库
- [ ] 网格交易策略实现
- [ ] DCA（定投）自动化
- [ ] 跨交易所套利监控工具
- [ ] 趋势跟踪策略

### 数据分析增强
- [ ] 更详细的交易归因分析
- [ ] 风险指标扩展（VaR、Sortino Ratio）
- [ ] 季度/年度绩效报告

---

## 🔮 长期目标 (2025年下半年及以后)

### 多智能体系统
- [ ] 将单体架构拆分为专业化智能体：
    - 风控智能体
    - 情绪分析智能体
    - 执行智能体
- [ ] 智能体间协作机制

### 机器学习集成
- [ ] 使用收集的数据训练自定义价格预测模型
- [ ] 基于本地 LLM 的情绪分析
- [ ] 异常检测与预警

### 移动应用
- [ ] iOS/Android 配套应用
- [ ] 推送通知
- [ ] 快速状态查看
