# 任务 / 资源清单

## 1. 云端环境

- 云服务器：阿里云 ECS（Ubuntu 24.04 LTS）
- Docker：运行中
- 青龙：运行中（单 Worker Pipeline）
- Redis：Docker 容器，已启用 AUTH
  - 宿主端口：26739
  - 本地映射端口：6379（SSH 隧道）
- SSH：ED25519 免密 + 端口 22

## 2. 本地 <-> 云端通信

本地通过 SSH 隧道访问云端 Redis（示例）：

```bash
ssh -N -L 6379:127.0.0.1:26739 root@<ECS_IP>
```

- 本地 Python（Anaconda / Python 3.13）可直接读写 Redis

## 3. 云服务器资源

- 阿里云：用于托管 MCP 项目，可能涉及计算、存储和网络资源。
- Redis：作为缓存和消息队列系统，用于存储会话数据、任务队列等。
- 青龙：青龙面板作为任务调度工具，帮助自动化任务管理。

## 4. API 资源

- OpenAI：用于调用 GPT 模型进行文本生成、对话管理等。
- Claude：作为 AI 提供商，支持多模态的对话和任务处理。
- Binance API：用于获取市场数据、执行交易等操作，主要通过测试网进行操作。

## 5. 第三方服务

- 飞书 Webhook：用于发送通知和报告，增强通信能力。
- Server酱：用于发送推送通知，将结果或警告推送到用户手机等设备。
- Notion：用于信息管理，任务和项目的可视化整理，方便团队协作。

## 6. 代码库与开源工具

- CCXT：用于加密货币交易所的接口管理，支持与 Binance 等多个交易所的集成。
- Python 库：如 Pandas、NumPy 等，用于数据处理和分析。

## 7. 项目目录结构

```text
/src
  ├── /tools            # 各类功能工具，例如市场数据获取、风险控制、交易执行等
  ├── /skills           # 复杂技能模块，例如自动交易策略、风险管理策略等
  ├── /core             # 核心业务逻辑，例如调度、MCP 主协议、工具注册管理
  ├── /storage          # 数据存储，例如交易记录、市场数据缓存、历史数据存储
  ├── /utils            # 工具类库，辅助功能（如日志、缓存、性能监控等）
  └── /tests            # 单元测试和集成测试：各模块拆分，允许迁移/合并现有模块
```
